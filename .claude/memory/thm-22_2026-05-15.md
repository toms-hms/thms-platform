- `CreateJobInput` in @thms/shared lacks `homeId`, so a local `type CreateJobData = CreateJobInput & { homeId: string }` was defined in jobs/mutations.ts rather than modifying shared types (out of scope).
- `JobImageDto` has no shared type; a local interface was defined in both components/jobs/queries.ts and components/jobs/mutations.ts capturing the API's augmented image shape (JobImage + url from presigned URL).
